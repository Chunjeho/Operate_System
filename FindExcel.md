# Find a name of the Excel column name

## Problem  

양의 정수가 주어 졌을 때, 주어진 수에 해당하는 엑셀 컬럼 이름을 구하시오.

When there is a given positive integer, find a name of the Excel column which is corresponding to that positive number.

### ex)1
Input: 1

Output: A
### ex)2
Input: 26

Output: Z
### ex)3
Input: 27

Output: AA
### ex)4
Input: 106

Output: DB
### ex)5
Input: 706

Output: AAD
